# Favify

A Spotify-inspired music app project with:
- `backend/`: Kotlin + Ktor backend API and static song assets
- `frontend-android/`: Android app built with Kotlin, Jetpack Compose, Retrofit, Room, and Hilt

## Notes
- Internal package names and historical project names are kept as-is.
- `local.properties`, build artifacts, IDE files, and git metadata were removed for GitHub upload.
- The Android app currently points to `http://10.0.2.2:8080/`, which is the Android emulator alias for the host machine.
